# Nexient - AI Job Agent

## Goal

A fully local AI-powered job search assistant that:

- Searches job postings
- Scores jobs against my experience
- Tailors my resume
- Generates cover letters
- Tracks applications
- Opens application pages for approval
- Uses LM Studio as the local AI
- Stores everything locally

## Technology

- LM Studio
- Python
- SQLite
- n8n
- Docker
- Playwright
- Git
